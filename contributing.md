# Contributing

## Setup

```sh
npm install
```

## Building

```sh
npm run build
```

## Pushing changes

ℹ️ The following commands only work for the project maintainer because they
require Google Drive authorization.

| Command               | Description       |
| --------------------- | ----------------- |
| `npm run push:gas`    | just GAS files    |
| `npm run push:js`     | just `sidebar.js` |
| `npm run push:static` | just HTML & CSS   |
| `npm run push`        | everything        |
